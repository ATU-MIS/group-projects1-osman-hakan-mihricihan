Plant Tracing System

Project Description

The **Plant Tracing System** is an academic project designed to **track plants and observe their growth and development over time**.  
The system allows users to record plant-related data and monitor changes during different growth stages.

This project aims to provide a structured way to understand plant development and improve plant care through continuous observation.



Project Objectives

- To track plant growth and development
- To record plant-related data over time
- To observe changes in different growth stages
- To support better plant care and monitoring
- To provide a simple and organized plant tracking system



Features

- Plant registration and tracking  
- Growth and development observation  
- Time-based data recording  
- Simple and user-friendly structure  
- Suitable for educational purposes
